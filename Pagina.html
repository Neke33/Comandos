Comando tail en Linux

El comando tail se utiliza para mostrar las últimas líneas de uno o varios archivos de texto en Linux.

Sintaxis:

tail [OPCIONES] [ARCHIVO]

Opciones:

  • -n, --lines=N: Muestra las últimas N líneas del archivo.
  • -f, --follow[={name|descriptor}]: Muestra las últimas líneas del archivo en tiempo real y sigue actualizando la salida mientras se añaden más líneas al archivo.

Comando head en Linux

El comando head se utiliza para mostrar las primeras líneas de uno o varios archivos de texto en Linux.

Sintaxis:

head [OPCIONES] [ARCHIVO]

Opciones:

  • -n, --lines=N: Muestra las primeras N líneas del archivo.
